# TikTok Ideas Dashboard

Public static dashboard for Braydon Carter's Hermes-native TikTok ideation engine.

Live URL: https://fred4599.github.io/tiktok-ideas-dashboard/

This repo publishes sanitized daily outputs only:

- daily TikTok idea cards
- competitor post/topic summaries
- Braydon performance notes
- early-catch AI/news opportunities
- coverage metadata for scrape/title/hook extraction

It intentionally does **not** publish credentials, raw Apify run metadata, Telegram metadata, private runtime files, or full local caches.
